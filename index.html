⚙ SEQTWO — First Time Setup
STEP 1 OF 4
CREATE A FIREBASE PROJECT

SEQTWO needs a free Firebase Realtime Database to store messages. This takes about 3 minutes.

  1. Go to console.firebase.google.com (sign in with Google)
  2. Click "Add project" → name it anything (e.g. seqtwo)
  3. Disable Google Analytics → click Create project
  4. Wait for it to create, then click Continue
STEP 2 OF 4
CREATE A REALTIME DATABASE
  1. In your project sidebar, click Build → Realtime Database
  2. Click "Create Database"
  3. Pick any server location → click Next
  4. Select "Start in test mode" → click Enable

⚠ Test mode allows open read/write for 30 days — enough to get started. You can add security rules later from the Firebase console.

STEP 3 OF 4
REGISTER A WEB APP & GET YOUR KEYS
  1. Click the gear icon ⚙ → Project settings
  2. Scroll to "Your apps" at the bottom
  3. Click the </> Web icon
  4. Give it any nickname → click Register app
  5. You'll see a firebaseConfig block — copy those values

You only need: apiKey, authDomain, databaseURL, and projectId.

ABOUT SEQTWO

SEQTWO is an anonymous messaging system. Each user gets a numeric ID.

Anyone who knows your ID can send you a message. You never reveal who you are.

HOW IT WORKS:
1. Your ID is generated and saved in your browser.
2. Share your ID with anyone.
3. They type your ID in Compose and send.
4. You see it in Inbox next time you open the page.
BACKEND: Firebase Realtime Database (your own project).
PRIVACY: No login. No tracking. IDs are random numbers.
